## Problem

**https://leetcode.com/problems/maximum-depth-of-binary-tree/**

## Solution

- Use recursion and do 1+max(left,right).

## Time Complexity

`O(N)`

## Space Complexity

`O(1) Extra Space + O(H) Recursion Stack space, where “H” is the height of the binary tree.`
